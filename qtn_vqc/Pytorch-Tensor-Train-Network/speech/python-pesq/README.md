# PyPESQ (WIP)
Pypesq is a python wrapper for the PESQ score calculation C routine. It only can be used in evaluation purpose.

## INSTALL
```
python setup.py install
```

## HOW TO USE
```python
import soundfile as sf
from pypesq import pesq

ref, sr = sf.read(...)
deg, sr = sf.read(...)

score = pesq(ref, deg, sr)
print(score)
```
